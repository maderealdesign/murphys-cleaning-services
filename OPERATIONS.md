## Deployment
Static HTML site. Publish directory: root folder. No build command required.

## Agent setup checklist
1. Push files to GitHub.
2. Link GitHub repo to Netlify.
3. Confirm live URL.
4. Update sitemap and canonical URLs to the final custom domain.
5. Add Telegram chat route in Agentify admin.
6. Run /setup in the Telegram chat.
